# RosterMate på macOS

## Nem installation uden terminal

Der er to muligheder:

1. Brug en simpel .app-bundle med Python og Flask indbygget
2. Brug en .pkg-installer, som installerer appen på computeren

I denne første version har jeg lavet et praktisk grundlag til at gøre det nemt at køre appen lokalt på macOS.

## Hvad du skal gøre

- Download repoet fra GitHub
- Dobbeltklik på install.command
- Rediger .env med dine oplysninger
- Start appen med run.command

## Fremtidig forbedring

Næste skridt er at pakke appen til en rigtig macOS-app (.app) eller en .pkg-installer, så du ikke behøver at bruge terminalen overhovedet.
