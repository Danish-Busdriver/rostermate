# Enkel workflow for RosterMate

## Kort version
- Brug beta som din egen test-version
- Brug main som den stabile version, som du deler videre
- Andre får kun main

## Sådan bruger du det

### 1. Start med beta
Brug beta på din egen maskine, når du tester nye ting.

### 2. Test alt grundigt
Prøv appen, kontroller at den virker, og tjek om noget er gået i stykker.

### 3. Når det virker, flyt det til main
Når du er tilfreds, overføres ændringerne til main.

### 4. Del kun main
Først når det er i main, kan du betragte det som klar til andre.

## Praktisk regel
> Beta er til dig. Main er til andre.
